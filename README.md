# Modele 3D - Szklarnia
W tym repozytorium umieszczone są również zaprojektowane elementy, które wydrukowano w technologii 3D na potrzeby projektu studenckiego, w którym zbudowano szklarnię na egzotyczne rośliny.

Znajdują się tutaj m.in. uchwyty na elektronikę (Raspberry Pi, ESP) oraz elementy montażowe. W razie potrzeby można pobrać zamieszczone pliki `.stl`, aby je modyfikować i wydrukować na własne potrzeby.
