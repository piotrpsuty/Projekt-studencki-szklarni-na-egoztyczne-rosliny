
# Modele 3D - Szklarnia

W tym repozytorium umieszczone są zaprojektowane elementy, które wydrukowano w technologii 3D na potrzeby projektu studenckiego, w którym zbudowano szklarnię na egzotyczne rośliny.

Projekt wykonano na Wydziale Fizyki Uniwersytetu Warszawskiego (UW).

Znajdują się tutaj m.in. uchwyty na elektronikę (Raspberry Pi, ESP) oraz elementy montażowe. W razie potrzeby można wykorzystać zamieszczone pliki `.stl`, aby wydrukować potrzebne elementy.
